# Minecraft Java Server Info Guide

Explore real-time information about a Minecraft Java server using the `/java` command. This guide will provide you with essential details about a Java server, including online players, player capacity, server version, and the MOTD (Message of the Day).

## Retrieving Java Server Info

In Minecraft Java Edition, you can access comprehensive information about a server using the following command:

```bash
/java [server-ip] [port]
```

## Information Provided

The `/java` command provides you with the following real-time server information:

- **Online Players:** The current number of players online on the server.
- **Player Capacity:** The maximum number of players that the server can accommodate.
- **Server Version:** The version of the Minecraft Java Edition server software being utilized.
- **MOTD (Message of the Day):** The custom message displayed to players when they connect to the server.
- **Server Icon:** The icon displayed next to the server in the server list.
- **Server Ping:** The latency between the client and the server.
- **Mod List:** The list of mods installed on the server.

## Example

To illustrate, let's say you want to retrieve real-time information about a server with the IP address "123.45.67.89" and port "25565." You would use the following command:

```bash
/java xxx.xxx.xxx.xxx | play.domain.com 25565
```
